# colorme-add-to-cart
## About
- 環境変数ACCOUNTで指定されたカラーミーショップ店舗の商品を直接カートに投入するリンクを構成するリダイレクタです

## Usage
環境変数としてACCOUNTにカラーミーショップのIDを指定しリダイレクタを起動します。

- https://(デプロイ先)/(カラーミーショップ商品ID)

上記リンクを経由することで該当リンクから直接ショッピングカートへ商品を投入することができます。
